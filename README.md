# Metadata Collector

This template should help get you started developing with Vue 3 in Vite.


## Project Setup

```sh
npm install
npm install uuid
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile, Minify, Format, and Run

```sh
npm run build;
npm run format;
npm run dev
```

### Localhost Link it Opens on
```sh
http://localhost:5173/
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
